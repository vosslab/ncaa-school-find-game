# USAGE.md

How to run the NCAA School Find Game tools.

## Local development server

Builds `dist/` and serves it on a random port, then opens the browser automatically:

```bash
./run_web_server.sh
```

## Build for GitHub Pages

Compiles TypeScript sources in `src/` and writes browser-ready output to `dist/`:

```bash
./build_github_pages.sh
```

## Run all codebase checks

Runs typecheck, typecheck:lint, lint, format:check, and node tests:

```bash
./check_codebase.sh
```

## Run node tests only

```bash
node --import tsx --test 'tests/test_*.mjs'
```

## Run Playwright tests

```bash
npx playwright test
```

## Data inputs (hand-edited)

These files are the authoritative raw inputs. Edit them to change school data or map geometry:

| File | Description |
| --- | --- |
| `data/ncaa_schools-FBS.csv` | FBS school roster (136 schools) |
| `data/ncaa_schools-FCS.csv` | FCS school roster (126 schools) |
| `data/ncaa_schools-NonFB.csv` | Non-football school roster (99 schools) |
| `data/us_states.geojson` | US state boundary GeoJSON |
| `data/school_coordinates_cache.json` | Geocoding cache (Nominatim results) |
| `data/school_colors_cache.json` | School color cache (Wikipedia lookups) |

## Generated artifacts (do not hand-edit)

These files are regenerated by the Python generators below. Do not edit them directly:

| File | Generated by |
| --- | --- |
| `src/data/schools_data.ts` | `build_school_data.py` |
| `data/schools.json` | `build_school_data.py` |
| `src/data/map_paths_data.ts` | `generate_map_paths.py` |
| `data/map_paths.json` | `generate_map_paths.py` |

The authoritative source for map state paths is `src/data/map_paths_data.ts` and
`data/map_paths.json`.

## Running the generators

Regenerate school data after editing the CSV files:

```bash
source source_me.sh && python3 build_school_data.py
```

Regenerate map state path data:

```bash
source source_me.sh && python3 generate_map_paths.py
```

Generate per-tier debug maps (written to repo root as `debug_map_*.html`; gitignored):

```bash
source source_me.sh && python3 generate_debug_map.py
```

`generate_debug_map.py` reads `data/schools.json` and `data/map_paths.json`. Run
`build_school_data.py` and `generate_map_paths.py` first if the JSON files are stale.

## Python test suite

```bash
source source_me.sh && python3 -m pytest tests/ -q
```
