# NCAA School Find Game

A browser-based geography quiz that challenges players to locate NCAA schools on a US map. Players click on state outlines to identify where colleges belong, covering Power and Mid-Major conferences with a tiered scoring system.

## Quick start

Open [ncaa_school_find_game.html](ncaa_school_find_game.html) in any modern web browser. No server or installation required.
